## Root Me Web Scraper

This script allows you to retrieve statistics on Root Me users.
# Installation
```bash
pip install -r requirements.txt
```
# Usage

Grant permissions to the script:
```bash
chmod u+x bin/rootme.py
```
Using the script:
```bash
./bin/rootme.py <csv>
```
Example of usage:
```bash
./bin/rootme.py /data/users.csv
```
